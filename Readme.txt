This interactive script aims to clean Brave of any unnecessary options to improve your security and privacy.

Ce script interactif à pour but de nettoyer Brave de toute options inutiles pour améliorer votre sécurité et vie privée.



EN : When you first launch PowerShell 7, do a "-SelfTest" to check that the commands are working correctly. then launch it normally, you arrive in an interactive menu. if you wish you can launch the simulation (DryRun) to see what will be applied. Choose option n°1 to apply the modifications, and option n°7 which will generate an HTML report with all the details and explanations of the Brave rules that have been modified

FR : Au premier lancement dans PowerShell 7  faite un "-SelfTest" pour contrôler que les commandes fonctionne bien. ensuite lancer le normalement, vous arriver dans un menu interactif. si vous le souhaitez vous pouvez lancer la simulation (DryRun) pour voir ce qui sera appliqués. Choisisser l'option n°1 pour appliquer les modifications, et l'option n°7 ce qui va généré un rapport HTML avec tout les détails et explications des règles de Brave qui ont été modifiées
